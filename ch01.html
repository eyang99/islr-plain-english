Chapter 1

Introduction

What "statistical learning" even means, what problems it solves, and the notation that will follow you for the rest of the book.

Big idea

Statistical learning = teaching a machine to find patterns in data.

You have inputs (things you can measure) and optionally outputs (things you want to predict). Statistical learning is the toolbox for building a mathematical guess about how the inputs relate to the outputs — a guess you can apply to new, unseen data.

The two big families

Supervised learning — you have labels

You have input/output pairs. You want the machine to learn the mapping so it can predict outputs for new inputs.

  • Regression — output is a number. Predict tomorrow's temperature, a house price, a salary.
  • Classification — output is a category. Is this email spam? Is this image a cat? Will this customer churn?
Supervised learning is like studying for a test using past exams with answer keys. You see the question, you see the answer, you learn the pattern.

Unsupervised learning — no labels, just data

You have inputs but no "right answer." You want the machine to find structure anyway — clusters, low-dimensional patterns, associations.

  • Clustering — group similar observations. "Which customer segments exist in my data?"
  • Dimensionality reduction — take 500 messy columns and distill them into 3 meaningful ones.
Unsupervised learning is being handed a shoebox of unlabeled photos and asked to organize them. Nobody told you the categories. You invent them.

A third cousin: semi-supervised

Some labels, mostly not. Real life. ISLR mostly skips this, but it's where most modern ML actually lives.

Notation you can't avoid

The book sticks to a consistent notation. Learn it once, save hours later.

SymbolMeaningExample
$n$Number of observations (rows)You have 1000 customers → $n = 1000$
$p$Number of predictors / features (columns)age, income, zip → $p = 3$
$X$The full $n \times p$ input matrixYour spreadsheet minus the target column
$x_{ij}$Observation $i$, predictor $j$Customer 5's age
$x_i$One observation (a row as a vector)Customer 5's full record
$X_j$One predictor (a column as a vector)The full age column
$y_i$Output for observation $i$Did customer 5 churn?
$\hat{y}$Predicted output (the hat = "estimated")Our model's guess
$f$The true, unknown relationshipNature's formula, hidden from us
$\hat{f}$Our model's approximation of $f$Our best guess of nature's formula
$\epsilon$Irreducible noiseThe part nothing can predict

The fundamental equation: $Y = f(X) + \epsilon$. There's a true relationship $f$, plus noise $\epsilon$ we can never capture. Our job is to estimate $f$. The noise stays noise.

Why statistical learning matters

Two jobs: prediction and inference

Prediction

You only care about $\hat{y}$ being accurate. The model can be a black box. Examples: spam filter, product recommendation, image classification. Accuracy wins; interpretability doesn't matter.

Inference

You want to understand how X relates to Y. Which predictors matter? Positive or negative effect? Linear or not? Examples: does this drug lower blood pressure? Which ad channel drives sales? Interpretability matters; raw accuracy is secondary.

Same toolbox, different goals. A simple linear model is great for inference — every coefficient tells a story. A random forest may be more accurate but won't tell you much about why. Pick the tool for the job.

Flashcards

1
Define statistical learning in one sentence.
A set of tools for building a mathematical model of how inputs relate to outputs, so you can predict or understand new data.
2
Supervised vs unsupervised — one-line difference.
Supervised has known outputs (labels); unsupervised doesn't. Supervised learns a mapping; unsupervised finds structure.
3
Regression vs classification.
Regression predicts a number (continuous). Classification predicts a category (discrete). Same setup, different output type.
4
What's the difference between prediction and inference?
Prediction: only ŷ accuracy matters, black box fine. Inference: understanding which predictors matter and how; interpretability required.
5
What is ε in Y = f(X) + ε?
Irreducible error — noise that even the true f can't explain. Sets a floor on how well any model can do.
6
n vs p?
n = number of observations (rows). p = number of predictors (columns). "n >> p" is easy; "p > n" is the hard problem Ch 6 handles.
7
What does the hat in ŷ or f̂ mean?
"Estimated" or "predicted." A hat means "this is my guess based on data, not the truth."

Explain it back

You're given a spreadsheet with customer data and a column "did they buy." Is this supervised or unsupervised? Regression or classification? What's n and p?
A drug company wants to know which of 20 biomarkers predicts response to a new treatment. Is this a prediction or inference problem? Why does the distinction change which model you'd pick?
